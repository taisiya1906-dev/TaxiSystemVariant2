# Система управления беспилотными такси

## Описание
Многопоточное приложение для моделирования работы парка беспилотных такси в городе.

## Требования
- Java 8 или выше
- OpenIDE / NetBeans (или другая среда разработки)

## Функциональность
- Генерация случайных заказов с координатами отправления и назначения
- Распределение заказов диспетчером между свободными такси
- Многопоточная обработка - каждое такси работает в отдельном потоке
- Потокобезопасная очередь заказов (BlockingQueue)
- Синхронизация через ReentrantLock и synchronized
- Автоматическая остановка системы через 30 секунд

## Ключевые технологии
- Многопоточность (Thread)
- Потокобезопасные коллекции (BlockingQueue, CopyOnWriteArrayList)
- Синхронизация (ReentrantLock, synchronized)
- Паттерны проектирования: Producer-Consumer, Thread Pool
